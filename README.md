##  Overview

This repository contains my solutions to the **C++ Modules** from **CPP00 to CPP09**, part of the C++ curriculum at **42 Network**.

The goal of these modules is to learn **Object-Oriented Programming (OOP)** in **C++98**, progressively covering:
- Basic syntax
- Memory management
- Inheritance & polymorphism
- Exceptions
- Templates
- STL
- Algorithms and parsing

All exercises respect **42 rules and constraints**.
